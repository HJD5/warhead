1. Установи библиотеки:
   pip install -r requirements.txt

2. Запусти main.py:
   python main.py

3. В приложении:
   - Используй вкладку "Карта" для просмотра маршрута и зон
   - Добавляй дронов и пилотов через формы (можно расширить)
   - Для демо: маршруты и зоны можно редактировать в map_widget.py
   - База данных utm.db создаётся автоматически

P.S. Вся структура кода максимально простая — чтобы легко доработать под свои нужды.
